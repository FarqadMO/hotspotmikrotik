# Supercell Hotspot Login

Modern hotspot login pages for MikroTik routers. Built for Supercell x Halab collaboration.

## Features

- Arabic interface with RTL support
- Dark/light theme toggle
- Landing page with free connection option
- Clean, modern design
- Mobile-friendly

## Installation

Upload these files to your MikroTik router's hotspot folder:
- `login.html`
- `alogin.html` 
- `status.html`
- `logout.html`
- `error.html`
- `md5.js`
- `img/` folder with logos
- `fonts/` Folder with fonts
- `img/` folder with images

## Pages

- **login.html** - Main login with landing page
- **alogin.html** - Success page after connection
- **status.html** - Connection status widget
- **logout.html** - Logout confirmation
- **error.html** - Error messages

## Customization

Replace logo images in `img/` folder:
- `supercell-logo.png` / `supercell-logo-dark.png`
- `halab-logo.jpg`
- `icon-logo.png` (favicon)
- `mask-bg.svg` (decorative background)
